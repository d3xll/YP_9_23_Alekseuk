# Свойства и определение свойств
**Свойства** - это член класса как поле или метод. Свойства используются как описание метода. 

При описании свойства определяются два специальных метода, которые непосредственно связанны со свойством.
Эти методы называются ***аксессорами:***

* ```get``` вызывается при считывания значения свойства. 

* ```set``` используется когда **вызывается при присваивании значения свойству**

###### Структура аксессоров:
```
[модификаторы] тип_свойства название_свойства
{
    get { действия, выполняемые при получении значения свойства}
    set { действия, выполняемые при установке значения свойства}
}
```

* **Считывание** значения свойства означает вызов метода, возращающего значения.
* **Присванивания** значения свойству подрузумевает вызов метод, который короче заменяет поле.

Свойства могут быть открытыми и закрытыми.

```
Animal Gepard = new Animal(); // Создает экземпляр класса

Gepard.IsPredator = true; // Присваевает значение в экземпляре класса
Gepard.Print(); // Вызов метода Print() из экземпляра Animal

Console.WriteLine(Gepard.Name); // Вывод в консоль имя животного.

class Animal
{
    private string name = "Cheetos"; // Изначально класс с этим именем: Cheetos
    private bool isPredator = false; // Изначально класс имеет isPredator = false
    public bool IsPredator 
    {
        set { isPredator = value; } // Сеттер для изменения значения isPredator
    }
    public string Name
    {
        get { return name; } // Геттер (для чтения) для значения name
    }

    public void Print() => Console.WriteLine($"Name: {name}  isPredator: {isPredator}"); // Публичный метод для вызова в экземплярах типо
}
```

Блок init - блок инициализации.
