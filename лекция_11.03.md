# Базовые принципы ООП в C#
#### C# - полностью объектно ориентированный язык программирования.
На сегодня существует да основынх способа организации программного кода: ООП и процедурно-струкрутрное программироваоние.

При процедурном программировании программа реализуется как набор под программу (процедуры и функции) исп. для обработки данных.

**Объектно ориентированное программирование** - способ организации через взаимодействие отдельных объектов, содержащих данных и методы для работы с этими данными.

---

**Есть три базовых принципа:**
1. Инкапсуляция
2. Полиморфизм
3. Наследование

**Инкапсуляция** означает, что данные объединяются в одно целое с программным кодом предназначеным для их обработки.

**Полиморфизм** подрузамевает использование единного интерфейса для решения однотипных задач.

**Наследование** позволяет создавать объекты не на пустом месте, а с использованием ранее утилит.

# Классы и объекты
Описание класса начинается с ключевого слова ```class``` после ключевого слова ```class``` указывается название класса.

В теле класса описываются:
* Поля
* Методы

Обычно поля и методы записываются с ключевым словом определяющий уровень доступа. Это касается и переменных

Объектная переменная объявляется как и переменная базового типа, но только в качестве модификатора указывается имя класса.
###### Пример:
```MyClass obj = new MyClass();```

---

При обращении к полю объекта используется точечный синтаксис: указается имя объекта и послее неё название поля.

Методы нестатические вызываются также с использование точечного синтаксиса.
